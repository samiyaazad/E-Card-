# E-Card 💌

This project is an interactive **e-card website** created for the **Cœur YSWS** challenge.
It contains digital letters about figure skaters who inspired me while growing up.
Each card features a clickable envelope that opens to reveal a heartfelt message.

---

## 🌐 Live Demo

Visit the project here:
https://samiyaazad.github.io/E-Card-/

---

## 📖 How to Use

1. Open the website.
2. Choose a card.
3. Click the **heart/envelope** to open it.
4. The envelope animation reveals a personal letter about the skater.

The site works on **desktop and mobile browsers**.

---

## 🛠️ How I Made This Project

This project was built using:

* **HTML** for the structure of each card
* **CSS** for styling, layout, and envelope animations

Each skater has their own page with a custom message inside the animated envelope.

---

## ⭐ Challenges

One challenge was making the **envelope animation and layout work correctly**.
I had to carefully manage **z-index layers, positioning, and CSS transitions** so the letter would appear properly when interacting with the envelope.

Another challenge was **debugging the layout**, since small CSS issues could cause elements like the envelope or letter to disappear.

---

## 🎯 Inspiration

Growing up, I watched figure skating on TV and admired the dedication and passion of these athletes.
This project is my way of creating **digital letters about the skaters who inspired me**, and sharing that appreciation through code.
